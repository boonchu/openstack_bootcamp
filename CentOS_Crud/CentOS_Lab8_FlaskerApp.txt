Lab 9: Horizon Dashboard Installation\
The OpenStack dashboard, also known as Horizon, is a Web interface that enables cloud administrators and users to manage various OpenStack resources and services.\
Horizon can be used to manage instances and images, create key pairs, attach volumes to instances, manipulate Swift containers etc.\
The dashboard enables web-based interactions with the OpenStack Compute cloud controller through the OpenStack APIs.\
Step 1:\
SSH to AIO node with the credentials in Lab access.\
\
Enter following command and Type pass as the [root] password \
$su \
#source ~/openrc.sh\
Install Horizon\
Step 2:	\
Install the dashboard on the node that can contact the Identity Service as root. \
#yum install httpd memcached mod_wsgi openstack-dashboard \'96y\
(NOTE: httpd to configure web server)\
Step 3:\
Edit /etc/openstack-dashboard/local_settings and change OPENSTACK_HOST to the hostname of your Identity Service:\
#vi /etc/openstack-dashboard/local_settings\
      OPENSTACK_HOST = "aioX"\
Step 4:\
Start the Apache web server and memcached :\
#service httpd restart\
#service memcached restart\
Step 5:\
Open the web browser and type http://aio node IP address/horizon.\
\
Type user name as admin and password as pass}
